Installation Guide¶

Installation¶

Dependencies¶

mlstacking requires:

  • Python (>= 3.5)

  • NumPy (>= 1.11.0)

  • SciPy (>= 0.17.0)

  • joblib (>= 0.11)

User installation¶

If you already have a working installation of numpy and scipy, the easiest way to install mlstacking is using pip.

pip install mlstacking
